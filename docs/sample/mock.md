# Mock request

## Simple Demo

### Config

```yaml
name: proxy
description: proxy sample
resources:
  - path: '/api/v1/test-dubbo/mock'
    type: restful
    description: mock
    methods:
      - httpVerb: GET
        onAir: true
        mock: true
        timeout: 100s
        inboundRequest:
          requestType: http
```

### Request

```bash
curl localhost:8888/api/v1/test-dubbo/mock -X GET 
```

### Response

```json
{
    "message": "mock success"
}
```
